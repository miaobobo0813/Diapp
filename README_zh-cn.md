# Diapp

[English](./README.md) 简体中文

一个注重隐私保护的个人日记软件。

## 功能与特色

- **现代化界面**：使用 WinUI 3 框架开发，界面美观，性能优异。
- **极致轻量化**：安装后应用体积仅约 11.9 MB。
- **高度安全**：
    - 支持为每篇日记设置独立密码。
    - 自定义加密算法，结合用户名、密码等多因子加密。
    - 不限制密码最大长度，可设置超高强度密码。
- **便捷安装**：采用 MSIX 格式，提供一键式安装体验。
- **完全开源**：采用 MIT 许可证，代码完全公开。

## 安装说明

1.  进入 `Installer` 文件夹，选择与您系统架构（如 x64, ARM64）对应的子文件夹。
2.  **首先安装证书**：双击 `Diapp_2.1.0.0_[架构].cer` 证书文件，并按照提示将其安装到“受信任的根证书颁发机构”存储区。
3.  **安装应用**：双击 `Diapp_2.1.0.0_[架构].msix` 文件进行安装。

## 技术栈

- **开发框架**: WinUI 3, C#
- **运行环境**: Windows 10 2004 (Build 19041) / Windows 11
